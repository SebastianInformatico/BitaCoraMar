# BitaCoraMar

Aplicacion Expo/React Native para gestion basica de registros.

## Requisitos

- Node.js LTS
- npm (o yarn/pnpm)
- Expo Go en el telefono (opcional)

## Instalacion

```bash
npm install
```

## Uso

```bash
npm run start
```

Luego:
- Android: `npm run android`
- iOS: `npm run ios`
- Web: `npm run web`

Para limpiar cache:

```bash
npm run start:clear
```

## Estructura

- src/screens: pantallas
- src/database: consultas y acceso a datos
- src/services: servicios de soporte

## Scripts utiles

- `npm run doctor` valida el entorno de Expo
